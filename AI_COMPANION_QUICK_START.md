## ğŸ› COMMON ISSUES & FIXES

### "Voice recognition not supported"
**Cause**: Browser doesn't support Web Speech API
**Fix**: Use Chrome, Edge, or Safari (not Firefox)
**Workaround**: Use text input

### No sound from AI
**Cause**: Browser sound settings
**Fix**: Check volume, unmute browser tab
**Workaround**: Read text responses

### Weather not loading
**Cause**: No internet connection
**Fix**: Connect to internet
**Note**: Everything else works offline

### Conversations not saving
**Cause**: localStorage disabled
**Fix**: Disable private/incognito mode
**Note**: Check browser storage settings
